# compute-grid-sdk

Volunteer compute grid with task verification and reputation-weighted scheduling.

## Languages
rust, python, go

## Interfaces
- `TaskScheduler`
- `WorkerPool`
- `ResultVerifier`
- `ReputationTable`

## Protocols
- `boinc`
- `folding`
- `seti`

## Build

```bash
cargo build --all-features
```

# cross-chain-bridge-sdk

Trust-minimized bridge with light-client verification and relayer auctions.

## Languages
rust, solidity, go

## Interfaces
- `LightClient`
- `RelayerAuction`
- `Lockbox`
- `Prover`

## Protocols
- `ibc`
- `layerzero`
- `wormhole`

## Build

```bash
cargo build --all-features
```

# mesh-network-sdk

Self-healing mesh topology with OLSR-inspired routing for IoT and edge clusters.

## Languages
rust, python, c

## Interfaces
- `MeshNode`
- `RoutingTable`
- `PacketForwarder`
- `TopologyMonitor`

## Protocols
- `olsr2`
- `babel`
- `802.11s`

## Build

```bash
cargo build --all-features
```

# p2p-content-sdk

Content-addressed peer-to-peer distribution with Merkle verification.

## Languages
rust, typescript, go

## Interfaces
- `ContentStore`
- `PeerManager`
- `MerkleVerifier`
- `BitswapEngine`

## Protocols
- `libp2p`
- `bitswap`
- `graphsync`

## Build

```bash
cargo build --all-features
```

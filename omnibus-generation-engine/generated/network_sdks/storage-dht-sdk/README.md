# storage-dht-sdk

Mutable DHT with erasure-coded storage and proof-of-retrievability.

## Languages
rust, go, python

## Interfaces
- `DHTNode`
- `ErasureCoder`
- `PoRVerifier`
- `Republisher`

## Protocols
- `kademlia`
- `s-kademlia`
- `pastry`

## Build

```bash
cargo build --all-features
```

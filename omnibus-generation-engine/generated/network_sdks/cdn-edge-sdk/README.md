# cdn-edge-sdk

Edge-compute CDN with cache invalidation and push-based prefetch.

## Languages
rust, go, typescript

## Interfaces
- `EdgeCache`
- `OriginProxy`
- `Prefetcher`
- `Invalidator`

## Protocols
- `http3`
- `quic`
- `srt`

## Build

```bash
cargo build --all-features
```

# iot-sensor-mesh-sdk

Low-power sensor mesh with time-synchronized data fusion.

## Languages
c, rust, micropython

## Interfaces
- `SensorNode`
- `DataFusion`
- `PowerManager`
- `Timesync`

## Protocols
- `mqtt-sn`
- `coap`
- `lora`

## Build

```bash
cargo build --all-features
```

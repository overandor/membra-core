# anonymous-mixnet-sdk

Layered encryption mixnet with Sphinx packet format and cover traffic.

## Languages
rust, go, python

## Interfaces
- `MixNode`
- `SphinxBuilder`
- `PathSelector`
- `CoverTraffic`

## Protocols
- `sphinx`
- `loopix`
- `nym`

## Build

```bash
cargo build --all-features
```

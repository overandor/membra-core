# dag-ledger-sdk

Directed acyclic graph ledger for high-throughput asynchronous consensus.

## Languages
rust, go, python

## Interfaces
- `GraphStore`
- `TipSelector`
- `ConflictResolver`
- `Snapshotter`

## Protocols
- `iota`
- `hashgraph`
- `phantom`

## Build

```bash
cargo build --all-features
```

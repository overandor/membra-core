# federated-learning-sdk

Decentralized ML training with differential privacy and secure aggregation.

## Languages
python, rust, cpp

## Interfaces
- `Aggregator`
- `LocalTrainer`
- `PrivacyFilter`
- `ModelExporter`

## Protocols
- `fedavg`
- `fedprox`
- `scaffold`

## Build

```bash
cargo build --all-features
```

# streaming-media-sdk

Adaptive bitrate streaming with WebRTC mesh and SRT relay fallback.

## Languages
rust, cpp, typescript

## Interfaces
- `MediaPipeline`
- `ABRController`
- `WebRTCMesh`
- `SRTRelay`

## Protocols
- `webrtc`
- `srt`
- `dash`

## Build

```bash
cargo build --all-features
```

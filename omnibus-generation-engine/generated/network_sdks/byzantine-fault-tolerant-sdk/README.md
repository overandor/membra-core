# byzantine-fault-tolerant-sdk

BFT state machine replication with threshold signatures.

## Languages
rust, go, cpp

## Interfaces
- `Replica`
- `ClientProxy`
- `ThresholdSigner`
- `ViewChanger`

## Protocols
- `pbft`
- `sbft`
- `minbft`

## Build

```bash
cargo build --all-features
```

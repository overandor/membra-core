# identity-did-sdk

Self-sovereign identity with verifiable credentials and selective disclosure.

## Languages
rust, typescript, python

## Interfaces
- `DIDResolver`
- `CredentialIssuer`
- `SelectiveDisclosure`
- `RevocationRegistry`

## Protocols
- `did-core`
- `vc-data-model`
- `oidc4vc`

## Build

```bash
cargo build --all-features
```

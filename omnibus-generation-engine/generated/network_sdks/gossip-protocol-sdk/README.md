# gossip-protocol-sdk

Epidemic gossip with anti-entropy and vector-clock reconciliation.

## Languages
rust, erlang, go

## Interfaces
- `GossipAgent`
- `StateReconciler`
- `MembershipList`
- `AntiEntropy`

## Protocols
- `swim`
- `plumtree`
- `scuttlebutt`

## Build

```bash
cargo build --all-features
```
